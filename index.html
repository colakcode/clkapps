Android Libraries


clkPieCharts


The clkPieCharts library allows you to easily create stylish pie and donut graphics in your Android application. If you want to get data from the user in your app and show this data in a pie or donut chart, you can use the clkPieCharts library. At the same time, get detailed views by clicking the slices. Please check your github description for details of the library. Click for clkPieCharts





clkProgress


Use the clkProgress library for a more stylish, useful and customizable progress tool in your Android projects. With the clkProgress library, you can have a customizable CircularProgress and a ProgressDialog where you can inform users. Click for clkProgress





clkOptionsDialog


Easy Options Dialog for your Android app. Dialog window with three option buttons for simple requests you want to users in your Android applications.
